Instructions

  • Download the .zip file
  • Extract contents
  • Open the extracted contents in IntelliJ
  • Go to File > Project Structure, and select SDKs
  • Click the ‘+’ sign to add an SDK, and select Download SDK
  • In the pop-up menu, select version 17, and vendor BellSoft Liberica SDK. Download the SDK
  • Select the downloaded SDK in the SDK menu, click Apply, then OK
  • Run the “LostCitiesApplication.java” file
